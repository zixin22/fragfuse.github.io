FragFuse

Memory-Based Query Fragmentation Attack on LLM Agents
Zixin Rao, Ying Wu, and Co-authors
Code Extended Version Paper

Abstract

FragFuse studies a memory-based query fragmentation attack against access-control mechanisms in LLM agents. The attack decomposes prohibited task intent into benign-looking fragments, stores them through normal memory interactions, and later retrieves and assembles them to bypass access-control checks while preserving the original task objective.

Overview

The project highlights a new attack surface in memory-augmented LLM agents: the interaction between long-term memory, retrieval, instruction assembly, and access-control enforcement. We evaluate the attack in controlled benchmark environments and discuss mitigation directions for safer agent deployment.
